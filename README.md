# real-gold-coin
任何有价凭证或有价票据都必须以实物做担保。
人类历史上最早的纸币是北宋的交子，它以实物铜钱或银币等作为担保。马克思认为货币天然是金银，挖矿需要人力和时间，与劳动价值论契合。
